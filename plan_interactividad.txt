  FUTBOL (FootballMatches_cleaned.csv) - 76,927 registros
  -------------------------------------------------------
  Columnas: MatchDate, HomeTeam, AwayTeam, HomeElo, AwayElo, Form3Home,
            Form5Home, Form3Away, Form5Away, HomeShots, AwayShots,
            HomeTarget, AwayTarget, HomeFouls, AwayFouls, HomeCorners,
            AwayCorners, HomeYellow, AwayYellow, HomeRed, AwayRed,
            OddHome, OddDraw, OddAway, FTHome, FTAway, FTResult
  Target: FTResult (H/D/A) -> Clasificacion de 3 clases
  Modelo entrenado: SI (best_model.h - Regresion Logistica, Acc: 61%)

  BASQUETBOL NBA (NBA_cleaned.csv) - 73,280 registros
  -------------------------------------------------------
  Columnas: gameDate, hometeamName, awayteamName, hometeamId,
            awayteamId, homeScore, awayScore, winner
  Target: winner (ID del equipo ganador) -> Clasificacion binaria
  Modelo entrenado: NO (pendiente)
  NOTA: Este dataset tiene pocas features. Se necesitara hacer Feature
        Engineering (crear columnas calculadas como racha, diferencia de
        puntos historica, ventaja de local, etc.) para que el modelo
        tenga suficiente informacion para aprender.

  BEISBOL MLB (mlb_elo_cleaned.csv) - 221,695 registros
  -------------------------------------------------------
  Columnas: date, season, team1, team2, elo1_pre, elo2_pre,
            elo_prob1, elo_prob2, rating1_pre, rating2_pre,
            rating_prob1, rating_prob2, pitcher1, pitcher2,
            score1, score2
  Target: Crear columna "winner" (team1 si score1>score2, else team2)
          -> Clasificacion binaria
  Modelo entrenado: NO (pendiente)
  NOTA: Excelente dataset con Elo y probabilidades pre-calculadas.
        Los pitchers (pitcher1, pitcher2) son un factor crucial en
        beisbol y se deben codificar como features.

      - Entrenar los 5 modelos
      - Guardar como models/best_model_mlb.h

  1.3 Modificar train_and_evaluate.py para que acepte un parametro

      - predict_mlb(team1, team2, pitcher1, pitcher2) -> {team1: 55%, team2: 45%}
